☕ StarCoffee Website

Projeto de website para uma cafeteria fictícia chamada StarCoffee, desenvolvido com HTML e CSS.
O site foi criado com o objetivo de praticar desenvolvimento web, layout moderno e responsividade.

🌐 Acesse o site

Clique no link abaixo para visualizar o site online:
👉 https://gabrielrosa-dev.github.io/starcoffee-site/index.html

📱 Funcionalidades

- Layout moderno inspirado em cafeterias reais
- Design responsivo (funciona no celular e computador)
- Página inicial com apresentação
- Catálogo de produtos
- Menu da cafeteria
- Página de contato

🛠️ Tecnologias utilizadas

HTML5
CSS3
GitHub Pages
